# 🎓 Student Grade Calculator (Python)

## 📖 About

This is a beginner-friendly Python project that calculates a student's:

- Total Marks
- Percentage
- Grade
- Pass/Fail Result

## 🚀 Features

- Accepts student name
- Accepts marks for 5 subjects
- Calculates total marks
- Calculates percentage
- Displays grade
- Displays pass/fail result

## 🛠️ Technologies Used

- Python 3

## ▶️ How to Run

1. Clone the repository
2. Open the project folder
3. Run:

```bash
python student_grade_calculator.py
```
